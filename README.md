# JetStyle test task 

## Usage
```
python object_arranger.py -h
```
```
python object_arranger.py 10000
```
```
python object_arranger.py 1000 -d ou -j json -i image
```

## Performance 
| Amount of objects |Execution time (in milliseconds) | 
| ----------------- |: -----------------------------: |
|               100 |                           2 - 8 |
|              1000 |                         14 - 20 |
|             10000 |                        80 - 110 |
